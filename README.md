# the_culture_of_international_relations
Repository for NLP-scripts related to the The Culture of International Relations project.

## Install

Use `pipenv install`

### Install labextensions

jupyter labextension install @jupyter-widgets/jupyterlab-manager jupyter-matplotlib jupyterlab-datawidgets jupyterlab_bokeh @jupyterlab/geojson-extension
